# ChatMe

Chats & Messages service for NEAR Blockchain

## Install

```bash
npm install --save chatme-widgets
```

## Usage

```tsx
import { ChatmeChat } from 'chatme-widgets'
```

```html
<ChatmeChat chatId="{}"
            network="testnet"
            chatBodyClass="" 
            connectButtonClass=""
            bottomBlockClass="" 
/>
```
